Posts2Excel is a simple plugin to backup your blog posts into a Excel Spreadsheet.

== Description ==

Posts2Excel is a very simple plugin which allows blog owners to backup all of their blog posts into a single Microsoft Office Excel spreadsheet.


== Installation ==


1. Upload the posts2excel folder into your wordpress plugin directory.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click on the admin menu for the plugin.
4. Push the Go button and your posts will be backed up.
